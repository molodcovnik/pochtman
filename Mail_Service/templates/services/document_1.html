Быстрый старт

API Почтмен позволяет принимать формы с ваших сайтов, приложений, ботов.

Используйте ваш API ключ для доступа к нашему API из ваших приложений или веб-сайтов. Следуйте инструкциям в документации API для получения доступа к доступным функциям и методам. API-ключ Вы можете получить на странице {% if request.user.is_authenticated %}профиля{% else %}профиля{% endif %}.

Подготовка

Чтобы начать работать с Почтмен, вам нужно зарегистрироваться и получить доступ к личному кабинету. Для аутентификации запросов в API вам потребуется секретный ключ и идентификатор формы из {% if request.user.is_authenticated %}личного кабинета{% else %}личного кабинета{% endif %}. Так же можете посмотреть наше короткое видео по использованию Почтмен.

Создание и настройка формы

Используйте наш интуитивно понятный {% if request.user.is_authenticated %}конструктор{% else %}конструктор{% endif %} форм для создания и настройки форм в соответствии с вашими потребностями.

  • Дать название форме
  • Выбрать необходимые поля
  • Сохранить форму
  • Укажите email в деталях формы, для получения уведомлений
  • Подпишитесь на нашего бота который рассылает уведомления
  • Укажите в деталях формы ваше имя пользователя в Телеграм

Интеграция формы

  • После настройки формы получите HTML код , вставьте его к себе на сайт
  • JS-код вставьте к себе на сайт внутрь тега <script></script>
  • В JS-коде вставьте ваш API-ключ в headers

В JS-коде не меняйте расположение полей, как наше приложение поставило поля так и оставьте их, чтобы не ломать их отображение в нашем приложении. В HTML-коде форму можно изменять как угодно, за исключением id-полей.
